## Project Overview
This project analyzes the last decade of Dow Jones price data to identify market downturns. The main goal was to identify downturns and categorise them.

## Key Insights
* **Downturns:** In total, there were 6 downturns, which started at a percentage change of more than 5% compared to the previous month. From which 5 were dips and 1 a crash.
* **Methodology:** Used Python (Pandas) for data cleaning and Matplotlib for visualizing the different downturns.

## Tech Stack
* **Language:** Python 3
* **Libraries:** Pandas, NumPy, Matplotlib
* **Environment:** Kaggle / Jupyter Notebook
